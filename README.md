# Weather App

A simple and clean React application that fetches and displays real-time weather data for any city using the OpenWeatherMap API.

## 🛠️ Tech Stack
* **Library:** React.js
* **Styling:** CSS3
* **API:** OpenWeatherMap API

## ⚙️ Key Functionalities
* Fetches real-time temperature and weather conditions.
* Dynamic search for global cities.
* Displaying humidity, wind speed, and weather descriptions.
